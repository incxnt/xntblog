充实 Shapefile 增强地图表现力

Shapefile 中囊括的地理信息以及其他国际标准信息使得制作网络地图的过程变成了一次轻松愉快的旅程. 同时 Shapefile 中往往并不包含自定义的个性化信息, 这些信息可以在生成 JSON 文件后再手工添加, 或利用 Map API 读取单独的文件并呈现在地图上, 但是上述的方法往往增加了不必要的工作量, 或使得 App 变得复杂, 并且增加了出错的几率.

为了解决上述问题, 可以将数据处理的工作提前至 Shapefile 文件阶段, 利用 GIS 桌面端应用添加必要的 属性字段以及相关的数据, 然后再生成 JSON 文件用于制作地图.

例如北京行政区划地图中使用的原始 Shapefile 中没有 行政区域地理面积以及人口数量的数据, 当将这些数据在制作地图之前就添加到 Shapefile 文件中后, 所生成的地图就可以包含丰富的个性化信息了.
